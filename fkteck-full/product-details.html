Features Technical Details Additional Information

Released in 2022 The XPS 9520 Laptop is Dell's latest release in the 15" XPS family and comes with all the latest technology including DDR5 Memory clocked at a whopping 4800MHz and a 12th generation Intel 14-core Processor. The Fourteen cores on the Core i9-12900HK processor allows for exceptional multitasking. To save power and prolong battery life the processor will run @ 3.8Ghz under low use but will boost to 5GHz under heavy use for powerful performance!

This XPS 9520 Laptop comes with a 15.6" OLED Touch Screen with 4K (3456x2160) resolution for stunningly clear visuals and a dedicated Nvidia GeForce RTX 3050 Ti graphics card with 4GB of Dedicated Graphics Ram great for gaming or photo and video editing

The 32GB DDR5 RAM Running @ 4800MHz will let you smoothly run multiple applications and browser tabs all at once and a 1TB (1000GB) PCIe NVMe M.2 Solid State Drive will provide fast startup, fast data transfers, and allow for large file storage!

Windows 11 is Microsoft's latest Operating System released in 2022 and if you want to downgrade to the older windows 10, you can downgrade with an easy download from Microsoft's website.The Home Edition will come with everything you need but you can also upgrade to windows pro from the windows store for $100.

With its 86Wh battery, numerous tests have shown that it can achieve nearly 10 hours of usage when web browsing with a screen brightness of approximately 150 nits. Compared to some other prominent names like the GIGABYTE AERO 15 or Lenovo IdeaPad Slim 7 Carbon, the Dell XPS 9520 offers longer battery life. However, when compared to the MacBook Pro 16 M1 Max, it falls short in this aspect.
